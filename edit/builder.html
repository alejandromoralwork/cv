Workflow

  1. Edit: Make changes in the CV editor (/edit/index.html)
  2. Build: Run the build script to generate static versions
  3. Test: Preview each version directory locally
  4. Deploy: Push only the version directories (not /edit/) to production
  5. Share: Send specific URLs to different companies

Build Command

Run this command in PowerShell from the /edit/ directory:

.\build-versions.ps1
